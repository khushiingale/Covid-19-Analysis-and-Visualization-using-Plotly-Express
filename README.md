# 🦠 COVID-19 Data Analysis & Visualization using Plotly Express

## 🚀 Overview
This project focuses on analyzing global COVID-19 data using Python and visualizing insights through interactive charts. It helps in understanding the spread, impact, and trends of the pandemic using powerful data visualization techniques.

---

## 🎯 Features
- 📊 Data cleaning and preprocessing  
- 📈 Exploratory Data Analysis (EDA)  
- 🌍 Country and continent-wise analysis  
- 📉 Cases, deaths, recoveries comparison  
- 🔵 Interactive visualizations using Plotly Express  
- 📊 Bubble charts, bar graphs, and line plots  

---

## 🛠️ Tech Stack
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Plotly Express  
- Google Colab  

---

## 📦 Installation & Setup
```bash
pip install pandas numpy matplotlib plotly
```

---

## ⚙️ How It Works
1. Load COVID-19 datasets (cases, deaths, recoveries)  
2. Clean and preprocess data (handle missing values, drop unused columns)  
3. Analyze data using Pandas  
4. Create visualizations using Plotly Express  
5. Interpret trends and patterns  

---

## 📊 Key Analysis
- Country-wise total cases, deaths, and recoveries  
- Continent-wise comparison  
- Time-series analysis of cases  
- Testing vs cases relationship  
- Case vs death correlation
